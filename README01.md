# Marketer Im Sumin Portfolio

퍼포먼스 마케터 임수민의 포트폴리오 웹사이트입니다.

## 🛠️ Tech Stack
- **HTML5**: 시맨틱 구조
- **CSS3**: Variables, Flexbox/Grid, Animations (No external libraries)
- **JavaScript**: Vanilla JS for interactions

## 📂 Project Structure
```
/
├── index.html      # 메인 파일 (내용 수정은 여기서!)
├── style.css       # 스타일 파일 (디자인)
├── script.js       # 인터랙션 (마우스 효과 등)
└── assets/         # 이미지 폴더 (사진은 여기에!)
```

## 🖼️ 이미지 변경 방법 (Image Guide)
포트폴리오에 들어가는 이미지를 바꾸고 싶으신가요?

1. **이미지 준비**: 사용할 이미지를 준비합니다.
2. **폴더에 넣기**: `assets` 폴더 안에 이미지를 넣습니다.
3. **이름 연결하기**: `index.html` 파일을 열어서 아래 부분을 찾아 파일명을 수정하세요.
   ```html
   <!-- 예시: InnerKurly 프로젝트 이미지 -->
   <img src="assets/원하는_이미지_이름.png" alt="InnerKurly Project">
   ```
   *팁: 이미지 파일명은 영문으로 하는 것이 오류가 적습니다! (예: `project_01.jpg`)*
